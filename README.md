# Main Repository for a Basic Controller

## Setup
Setup python repository

``` bash
python3.10 -m venv .venv
```

Install requirements:
``` bash
pip install -r requirements.txt
```

Install the python dependencies (for automatic development update):
``` bash
pip install -e .
```

## Compile
To compile, run:
``` bash
cargo build
```

Run 
``` bash
cargo run
```
